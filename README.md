# XVM Syntax Highlight

Extensão para realce de sintaxe e operadores do protocolo XVM no VS Code.

## Instalação

Instale pelo marketplace ou usando o arquivo `.vsix`.

## Funcionalidades

- Realce de comandos, variáveis e operadores matemáticos
- Paleta de cores customizável por operador

## Exemplo

```xvm
>VS1907_MAT,-40x5,+10/4,L10H50,x100L0H100<
